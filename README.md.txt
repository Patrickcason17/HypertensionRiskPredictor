# 🩺 Hypertension (BP) Risk Predictor

A machine learning web app that predicts a person’s 10-year risk of hypertension (high blood pressure) using data from the Framingham Heart Study.

---

## 🔍 Problem it Solves

High blood pressure is a silent killer. Early prediction can help prevent heart diseases and strokes. This tool helps users assess their risk level in just seconds.

---

## 🚀 Tech Stack

- **Python**
- **Pandas, Scikit-learn, Joblib**
- **Streamlit** for the web UI
- **Framingham dataset** (public heart study dataset)

---

## 🎯 Features

- Predicts risk using:
  - Age, Gender
  - Blood Pressure, Glucose
  - Cholesterol, BMI
  - Smoking status and more
- User-friendly sliders
- Instant prediction (High Risk / Low Risk)
